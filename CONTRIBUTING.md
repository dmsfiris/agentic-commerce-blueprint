# Contributing

This repository is a compact reference implementation for *The Agentic Commerce Blueprint*.

Changes should preserve:

- reason colocality;
- action-aware decision basis;
- eligibility-versus-authority separation;
- v4 contract and schema alignment;
- content-addressed rule sets;
- detached authenticator semantics;
- evidence hash-pinning;
- freshness horizons;
- generated-claim capability gates and inherited refusal;
- projection consumption of canonical meaning rather than local reinterpretation.

Run before submitting:

```bash
npm run check
npm run write:examples
```

Generated examples should change only when the canonical behavior changes.
